傅里叶和小波分析-Chapter1

Veröffentlicht am 2016-07-31   |   in Fourier and Wavelet

傅里叶级数

在高等数学当中我们都学过傅里叶级数，从数学空间来讲，傅里叶级数就是将一个目标函数$f(x)$ 在标准正交集空间的展开，而这个标准正交集的空间就是由三角函数构成的正交空间
\begin{equation}
\{\cos kx, \sin kx| k \in \mathbb{Z} \}
\end{equation}

正交空间

那什么叫正交空间，我们日常见得三维坐标轴其实就是一个正交空间，我们不妨假设一个标准的三维正交坐标轴为$X = [1, 0, 0], Y = [0, 1, 0], Z = [ 0, 0, 1]$。我们发现
\begin{equation*}
X \vec{Y} = 1 \times 0 + 0 \times 1 + 0 \times 0 = 0 \\
Y \vec{Z} = 0 \times 0 + 1 \times 0 + 0 \times 1 = 0 \\
X \vec{Z} = 1 \times 0 + 0 \times 0 + 0 \times 1 = 0
\end{equation*}
从而说明$X,Y,Z$三个单位坐标是相互正交的。上面是针对三维空间的标准正交定义，可以将其推广到高维空间来论证标准正交，在这里就不再赘述了。我们再将其推广到泛函空间,假设我们现在有一个函数集合$F = \{f_1(x), f_2(x),\dots, f_n(x)\}$,那么我们称该函数集合是标准正交集合如果它满足如下条件：
\begin{equation}
f = 1
\end{equation}
这个标准正交成立条件的第二个函数加了共轭，这是为了保证对于复数空间$\mathbb{C}$时，当$i = j$时，该积分表达式表达的是函数$f_i(x)$的长度需要其为一个正数，所以需要加一个共轭。根据这个定义我们容易证明$\{\cos kx, \sin kx| k \in \mathbb{Z} \}$就是一个标准正交空间。同样也容易证明$\{exp\{-inx\} | n \in \mathbb{Z}\}$也是一个标准正交空间。有了标准正交集之后我们就希望自己的目标函数$f(x)$能够在标准正交集上展开表示，即希望获得如下的表达形式：
\begin{equation}
f(x) = a_0 + \overset{\infty}{\underset{k=1}{\sum}} a_k \cos kx + b_k \sin kx
\end{equation}
对于系数的求解就是利用标准正交基的性质进行求解，也就是常见的傅里叶系数求解方式，在这里就不再多说了，直接给出傅里叶系数的求解公式：
\begin{equation}
a_k = \frac{1}{2\pi}
\end{equation}
这个傅里叶系数求解方式看着完美但是在实际应用当中是不现实的，因为我们通常用计算机进行傅里叶变换，而这个系数包含积分，不适合在计算机当中应用（当然如果大家度过数值计算的话那就另说了，如果是通过数值计算的方式的话，确实可以搞定这个傅里叶形式的积分，但我们再这里不考虑）
